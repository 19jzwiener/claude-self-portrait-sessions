# Profiles — Active Profile Pages

> Each file in this directory is a current-state profile page describing one dimension of the user's life. The runner reads these at the start of every session to know who this person is *today*.
>
> **Active vs inactive:** profile pages currently in use live here. Pages that have been deactivated for this user (because the dimension stopped being relevant, or the schema changed) live in `../inactive-profiles/` along with their change log.
>
> **History:** evolution of each page over time is tracked in `../changes/profiles/[page-name].md`. The pages themselves stay clean — current state only. Past values live in the change log directory.
