# Minutes — Per-Session Synthesis

> Each file here is one session's narrative synthesis — what mattered, condensed.
>
> **Naming:** `sNN-YYYY-MM-DD.md` matching the raw record's session number.
>
> **Contents:** verbatim quotes worth preserving, decision points, emotional/state moments, open questions THEY had, the arc of the session.
>
> Minutes are read by the runner before subsequent sessions to remember context fast. They preserve narrative; they're not a transcript replacement.
>
> For verbatim records, see `../raw/`. For what shifted in profile pages, see `../changes/`.
