# Inactive Profiles — Archived but Preserved

> Profile pages here are no longer active for this user but have not been deleted. Reasons a page might end up here:
> - The dimension stopped being relevant for this user
> - The schema changed and the page was retired in favor of newer pages
> - The user explicitly retired this dimension
>
> When a page is moved here from `../profiles/`, its change log is also moved from `../changes/profiles/` to `../changes/inactive-profiles/`. They travel together.
>
> Inactive pages can be reactivated by moving them (and their change log) back.
