# Raw — Per-Session Verbatim Records

> Each subdirectory here holds the literal, unprocessed record of one session.
>
> **Naming:** `s01-YYYY-MM-DD/` for session 1, `s02-YYYY-MM-DD/` for session 2, etc.
>
> **Contents per session:**
> - `transcript.md` — every screen, every word the user typed, in order, with timestamps
> - `feedback.md` — verbatim post-session Q&A
>
> Raw records are write-once. Never edit a raw record after the session ends — it's the historical baseline.
>
> For synthesis (what mattered narratively), see `../minutes/`. For what got updated in the user's profile pages as a result, see `../changes/`.
